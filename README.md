# 设备说明

超级网关CLX_S20M魔改焊接了双2.5G网口，可直接用S20P无WIFI版本固件，根据个人喜好增删插件。

# 固件简要说明

固件每天早上5点自动编译。

固件信息里的时间为编译开始的时间，方便核对上游源码提交时间。

MEDIATEK系列、QUALCOMMAX系列、ROCKCHIP系列、X86系列。

# 目录简要说明

workflows——自定义CI配置

Scripts——自定义脚本

Config——自定义配置
